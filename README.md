## Installation
 [+] Install git & php
```
apt-get install git -y
apt-get install php -y
```
 [+] Clone this Repo
```
git clone https://github.com/cyberjeevi/IG-Scraper.git
```
 [+] Inside Repo Unzip 'vendor.zip'
```
unzip vendor.zip
```
#### Run the Tool
```
php igscraper.php
```
